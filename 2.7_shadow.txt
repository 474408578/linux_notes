2、What's flags '!!' and '*' mean second field in /et/shadow?
	Answer: '!!' mean is this account not set password, '*' mean's this account is be locked, inactive.
3、which mean the 5th field int /etc/shadow?
	Answer: The number of days before a password is going to expire (see the maximum password age above)
during which the user should be warned.
	 An empty field and value 0 mean that there are no password warning period.
