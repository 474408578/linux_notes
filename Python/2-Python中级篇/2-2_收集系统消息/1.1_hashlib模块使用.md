## hashlib
- hashlib是python的一个内建模块
- 如果要看一个对象的帮助，需要在后面加的括号，如
help(hashlib.md5)这个相当于看hashlib.md5这个方法或者函数的帮助。
help(hashlib.md5())加了()，相当于看对象的帮助。
看函数的帮助不需要加()。

- 下面完成一个计算md５值的pythontkyc
```
0 #!/usr/bin/env python
1 #-*- coding:utf8 -*-
2 
3 import hashlib
4 import sys
5 
6 def md5sum( f ):
7     md5 = hashlib.md5()
8     fd = open(f)
9     
10     while True:
			# 下面表示每次循环读４个字节，默认不给参数就全部读完
11         data = fd.read( 1024 * 4)
12         if data:
13             md5.update( data )
14         else:
15             break
16     
17     fd.close()
18     
19     return md5.hexdigest()
20 
21 if __name__ == "__main__":
	   try:
22         print md5sum( sys.argv[1] )
	   except IndexError:
	       print '%s must be append a argument.' % __file__ #这个内置变量，会打印出程序名，和sys.argv[0]效果一样
```

```

```
