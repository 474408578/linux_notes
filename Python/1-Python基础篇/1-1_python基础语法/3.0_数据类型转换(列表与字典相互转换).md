##序列如何转换为字符串
str.join()
```
In [7]: ':'.join(l)
Out[7]: 'a:b:c'

In [8]: s = 'abdcefg'

In [9]: l = list(s)

In [10]: l
Out[10]: ['a', 'b', 'd', 'c', 'e', 'f', 'g']

In [11]: ':'.join(l)
Out[11]: 'a:b:d:c:e:f:g'
```
##字典转换为列表
并不是所有的列表都可以转换为字典，但是所有的字典都可以转换为列表，因为字典是以键值对组成的，所以一定是可以转换为列表的，但是列表并不是所有的都是成对出现的
dic.items()
```
In [13]: dic = {'a':1, "b":'23', 'c':323}

In [14]: dic
Out[14]: {'a': 1, 'b': '23', 'c': 323}

In [15]: dic.items()
Out[15]: [('a', 1), ('c', 323), ('b', '23')]

In [16]: ldic = dic.items()

In [17]: ldic
Out[17]: [('a', 1), ('c', 323), ('b', '23')]

In [18]: dict(ldic)
Out[18]: {'a': 1, 'b': '23', 'c': 323}
```
