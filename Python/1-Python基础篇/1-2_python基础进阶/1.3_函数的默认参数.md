#函数的默认参数
如果在给参数的时候只给了一个参数，而且另外一个参数有默认的参数在函数声明中定义，那么没有给的那个参数就使用函数的默认参数
```
In [9]: def fun(x, y=100):
   ...:     print x,y
...:     

In [10]: fun(2)
2 100

In [11]: fun(2,3)
2 3

```
在函数调用过程中也可以fun(x=2,y=3)这种方式来伟参，但是这里的x，y必须要和形参中定义的一样，否则就会报错
```
In [12]: def fun(x=2, y=3):
   ....:     print x,y
....:     

In [13]: fun(x=3,y=100)
3 100
```
