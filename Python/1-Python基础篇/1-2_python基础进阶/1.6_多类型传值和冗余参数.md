##多类型传值
在函数调用中把元组中的元素作为参数

```
def func1(x, y):
	return x + y

t1 = (1, 2)

print func1(*t) """这里一定要注意无组元素的个数一定要和函数形参的个数一致，否则就会报错"""

def func2(x, y, z):
	return x + y + z
t2 = (1, 2, 3)

print func2(*t2)
print func2(100, *t1) """这样也是可以的，一定要保证参数的个数是一致的"""
```

###把字典的元素作为函数的参数
如果要让字典的元素作为参数需要注意以下几个因素
1、字典的key要和函数的形参一致
2、定义字典的时候，key的顺序可以不用和函数形参一致，因为字典本身就是无序的
3、key一定要加一个引号，因为不加python会认为这是一个变量
```
In [53]: dic1 = {'x':1, 'y':2, 'z':3}

In [54]: func2(**dic1)
6

In [55]: dic2 = {'a':1, 'b':2, 'c':3}

In [56]: func2( **dic2 )
---------------------------------------------------------------------------
TypeError                                 Traceback (most recent call last)
<ipython-input-56-e8f1f19e7e62> in <module>()
----> 1 func2( **dic2 )

TypeError: func2() got an unexpected keyword argument 'a'

```
##冗余参数
定义
def func(x, y, *args, **kwargs):
可以看到冗余参数的形式为，命名参数，元组，字典
如果元素多于命名参数，那么就会存入元组中，那么如何把元素存入字典中呢？
可以使用类型赋值的方式，见下例
